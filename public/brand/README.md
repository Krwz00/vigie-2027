# Logo Le Millénaire

Déposez le vrai logo ici pour remplacer la pastille « M » de repli :

```
public/brand/logo-millenaire.svg   (préféré)
public/brand/logo-millenaire.png   (si pas de SVG)
```

- Le header et le footer basculent automatiquement sur ce fichier dès qu'il
  est présent (SVG prioritaire, PNG en second, sinon fallback « M »).
- Format carré recommandé (le logo est rogné en cercle, `object-contain`,
  ratio préservé — pas de déformation).
- Version lisible sur fond navy (`#050f1e`).
- **Aucune image cassée** : si le fichier est absent, l'app reste sur la
  pastille « M » dorée, sans erreur.

> Le lockup reste : logo + wordmark « VIGIE 2027 » (Space Grotesk). Rien d'autre.
